# Todo List

Небольшое веб-приложение для просмотра списка задач с использованием публичного API JSONPlaceholder.
Тестовое задание для компании "МАКСИС АЙ ТИ"

## Использованные технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

## Запуск проекта

1. Скачать или клонировать репозиторий.
2. Открыть папку проекта.
3. Запустить файл `index.html`.

Для корректной работы можно использовать расширение **Live Server** в Visual Studio Code.


## Что можно улучшить

- Реализовать debounce для поиска.
- Добавить пагинацию.
- Сохранять выбранные фильтры в Local Storage.
